A bit about me


My name is Andronikos Glaros, and I am a university student studying Computer Science at Democritus University of Thrace. I’m currently interested in developing desktop applications with Java Swing, and I’m also expanding my knowledge in Java™. In addition, I have basic experience with C for console applications, Android app development using Kodular™ and the Linux commandline.



My interest in coding


From a young age, I’ve always had a passion for computers. My first creations were small presentations in Microsoft PowerPoint with hyperlinks and slides that mimicked a simple UI. In junior high school, I experimented with Microsoft Visual Basic to build basic GUI projects, but I stopped during high school. At university, I started programming with C as my first language, and in my second year, I discovered Java. Since then, Java has become my true passion.
